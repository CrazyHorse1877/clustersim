First, rename the file to .exe so it can be run. Run the program from the xtender/bin folder. It will not work without this.
When you run, you will be prompted, first press 1 then enter, this starts the simulator. Then press 2 and enter as this tells it to take off.
Then press 3 and enter and a whole lot of values will be spit out, these are the different telemtery values. When this is done you can press 4(and enter to stop the simulator and then press 6 and enter to quit.

DO NOT PRESS ANY OTHER KEYS. THESE WILL ATTEMPT TO ACCESS THE REAL AUOPILOT WHICH IS NOT CONNECTED. I do not take any responsibility for doing this without the autopilot connected.
